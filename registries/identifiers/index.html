Status of this Document

This document is subject to change at any time. The terms defined herein will never be removed, but may be deprecated.

Deprecated term are identified by the label [DEPRECATED]. These terms are no longer recommended for use.

Terms without a label are considered stable.

Requests for additions, modifications and clarifications can be made through the Issue Tracker. Pull requests are also welcome.

1. Overview

This document defines a registry of identifier schemes for use in the Package Document opf:scheme attribute.

2. Identifier Schemes

DOI

A Digital Object Identifier.

<dc:identifier opf:scheme="doi">urn:doi:10.1016/j.iheduc.2008.03.001</dc:identifier>
ISBN

An International Standard Book Number.

<dc:identifier opf:scheme="isbn">urn:isbn:9780000000001</dc:identifier>
JDCN

A Japan Digital Comic Number.

<dc:identifier opf:scheme="jdcn"></dc:identifier>
UUID

A Universally-Unique Identifier (also referred to as a GUID).

<dc:identifier opf:scheme="uuid">urn:uuid:A1B0D67E-2E81-4DF5-9E67-A64CBE366809</dc:identifier>
