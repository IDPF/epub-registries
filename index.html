EPUB Registries

EPUB Collection Roles Registry
Defines values for Package Document collection elements.
EPUB Identifier Schemes Registry
Defines identifier schemes for the dc:identifier and dc:source elements.
EPUB Optimized Publication Standards Registry
Identifies standards that Optimized Publications conform to.
EPUB Publication Types Registry
Defines publication identifiers for the dc:type element.
EPUB Subject Authorities Registry
Defines authorities for the dc:subject element.
