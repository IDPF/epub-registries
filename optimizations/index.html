1. Overview

This informative document lists standards for Optimized EPUB Publications, which are publications that have been optimized for a specific reading modality.

Refer to the EPUB Accessibility specification for more information about these kinds of publications, and how to include the standard identifiers in the Package Document metadata.

2. Standards

Inclusion of standards in this registry does not constitute endorsement of them by the IDPF, its board or its membership.

DAISY Guidelines for Navigable Audio-Only EPUB 3 Publications

Provides guidelines for creating audio-only EPUB Publications using Media Overlays.

Identified by the conformance IRI:
http://www.daisy.org/guidelines/epub/navigable-audio-only-epub3-guidelines.
