Overview

This document lists identifiers for Optimized EPUB Publications – publications that have been optimized for a specific reading modality.

Refer to the EPUB Accessibility specification for more information about these kinds of publications, and how to use these identifiers in the Package Document metadata.

Roles

http://www.daisy.org/guidelines/epub/navigable-audio-only-epub3-guidelines

Identifies EPUB Publications that conform to the DAISY Guidelines for Navigable Audio-Only EPUB 3 Publications.
